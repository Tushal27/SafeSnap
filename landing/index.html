How it works

Set up in under 2 minutes

Six simple steps from download to protection — no technical knowledge required.

1

Create a Parent Account

Sign up at the parent dashboard on any browser — desktop or mobile. Takes 30 seconds.

2

Generate a Pairing QR

Go to Devices on the dashboard and tap "Pair new device". A unique QR code appears — valid for 5 minutes.

3

Install the App

Download the SafeSnap APK on your child's Android phone. Enable "Install from unknown sources" when prompted.

4

Select "Child's Device"

Open the app, tap "Child's device" and then "Scan pairing code". Point the camera at the QR code on your screen.

5

App Scans in the Background

SafeSnap quietly checks photos on-device using AI every hour. No internet needed for scanning — everything stays local.

6

Parent Gets Alerted

When flagged content is detected, the parent dashboard shows an alert with severity level — never the actual image.

Built different

Privacy by design

Most parental controls upload photos to the cloud. SafeSnap never does.

On-device AI only

The MobileNet classifier runs entirely on the phone. No photo ever touches a server.

Metadata alerts only

Only a severity score and image hash are reported — never the image itself.

Hourly background scans

WorkManager schedules quiet scans every hour without draining the battery.

QR pairing — no code

Secure one-time token QR code links parent and child accounts in seconds.

Severity levels

Four tiers — Low, Medium, High, Critical — so parents know what needs immediate attention.

Real-time dashboard

Parent web dashboard with live WebSocket alerts, device management, and alert history.
